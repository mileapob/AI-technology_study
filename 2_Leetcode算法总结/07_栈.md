## 2.栈、队列、list的区别

###  栈（先进后出）
- 操作：`append()` 加末尾，`pop()` 删末尾
- 代码：
```python
stack = []
stack.append(1)
stack.append(2)
stack.pop()  # 返回 2
```

### 队列（先进先出）
操作：append() 加末尾，pop(0) 删开头

代码：
```python
queue = []
queue.append(1)
queue.append(2)
queue.pop(0)  # 返回 1
```
