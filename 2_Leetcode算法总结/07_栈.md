## 2.栈、队列、list的区别

**栈用list：**
```python
stack = []
stack.append(x)   # 入栈
stack.pop()        # 出栈（从尾部）
```

**队列用deque：**
```python
from collections import deque
queue = deque()
queue.append(x)    # 入队（从尾部）
queue.popleft()    # 出队（从头部）
```
